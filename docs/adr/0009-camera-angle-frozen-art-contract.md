# 相机角度冻结为美术契约（贴图允许角度锁死）

tile 美术管线的相机角度（pitch 35.26° / yaw -15°，manifest 单一真相）原本是"单点可改、重烘即跟随"
的活参数。引入 AI 生图贴图管线后，候选方案 A′（设计稿 → 确定性反投影 warp → UV 贴图）产出的
贴图与该角度互逆绑定——改角度不再是"重烘"，而是全量重新生图 + 重审设计稿。

**决定：角度冻结为契约。** 理由：① 该角度是 11 轮截图迭代亲自收敛的，全部既有烘焙 PNG 与概念图
气质长在它上面；② 即便没有 A′，改角度本来就意味着整批美术气质重审——"一键改角度"在程序上成立、
在美术上从来不便宜，冻结只是把隐性成本显性化；③ 冻结后才允许"角度锁死型资产"入库，换来对批准
设计稿的像素级忠实（往返恒等：warp 拉伸 × 同相机烘焙压扁 = 精确互逆）。

## Consequences

- README 中"改视角 → 重烘 → Godot 自动跟随"降级为仅对程序化材质资产成立；含生图贴图的资产改角度 = 重新生成。
- 地块贴图生成方案 bake-off（design_warp / design_uv_gpt / dual_canvas 三方案并行，Godot 内实测定主力）中，
  design_warp 的合法性依赖本契约。
