## B. 历代公认经典技能（设计亮点）

以下为跨 set 社区公认设计突出的英雄技能。每条含 **设计亮点**（独特机制说明），最终目标是给 inkmon 提供设计模式参考。

### Sion (Set 2 — Glacial/Shadow) — "Glory in Death / Decimating Smash"
效果: 蓄力 1 秒后猛砸地面，大范围 AoE 击飞 + 晕眩 2.5/3/6 秒，造成 125/200/500 AP 魔法伤害
设计亮点: **蓄力 + 高惩罚高回报** AoE 控制技能。蓄力期间可被打断的张力；低星时控制时长就显著（已是 2.5s），而 3 星 6 秒的夸张晕眩让 3 星具极高价值
复现难度: 中

### Aurelion Sol (Set 6 — Dragon / 后续多 set 回归) — "Voice of Light / The Skies Descend"
效果: 持续引导星界能量，每次技能伤害积累 Stardust，Stardust 越多大招 "The Skies Descend" 越强（无上限叠加）
设计亮点: **无限 scaling 蓄能** — Stardust 无上限，一场战斗内越蓄越吓人，后期成"不放大招就赢"vs"放了大招一炮带走"的收割机制
复现难度: 中-高（需要累积变量机制）

### Akali (Set 10 — K/DA) — "Unrelenting Flow"（基于 K/DA 灯光区域变身）
效果: 进入 K/DA 灯区时触发羁绊激活，Akali 变身为 K/DA ALL OUT Akali，技能基于 Shuriken Flip 连招（突进-斩击-突进）
设计亮点: **位置敏感变身** — 站在特定 hex 区域（K/DA 灯光）才能激活变身形态，引入强位置博弈；变身技能组合是连招式而非单次爆发
复现难度: 高（需要场地特殊 hex 机制）

### Samira (Set 6 — Inferno/Trinity) — "Flair / Style"
效果: 扇形 AoE 斩击造成 150/160/175% AD 物理伤害 + 减 5/10/15 护甲；无目标时改为锁定单体（返还 20 mana）
设计亮点: **连击评级（Style Rating）** — Samira 每次攻击/技能叠 Style 等级 E→S，每级改变技能表现和音效；设计上让"节奏感"成为游戏玩法而不仅是数值
复现难度: 中-高（需要叠加等级条 UI）

### Sy'fen (Set 7.5 — Whispers/Dragon/Bruiser) — 7 费单位特殊

## C. 通用机制（跨 set）

### C1. 羁绊（Traits）效果类型

以下是跨 set 反复出现的羁绊设计范式，以 Set 17 为主例（因为数据最完整），辅以历代经典。

#### 增伤类（damage amplification）
- **Sniper**（狙击）: 每多一格距离 +X% 伤害
  Set 17: (2) 18% DA + 2%/hex · (3) 24% + 3%/hex · (4) 28% + 4%/hex · (5) 32% + 5%/hex
  **设计亮点**：距离越远伤害越高，与摆位强绑定
- **Rogue**（潜行者）: 直接 AD/AP 加成 + 低血逃脱
  Set 17: (2/3/4/5) 15/30/45/60% AD AP；生命 <50% 时进入影子重定向敌方仇恨
- **Challenger**（挑战者）: 攻速 + 目标死亡后突进下一目标
  Set 17: (2/3/4/5) 15/22/30/40% AS；5 件套额外 dash-to-target +50% AS 2.5s
  **设计亮点**：击杀奖励=连锁屠杀（类似"多杀奖励"）
- **Eradicator**（消除者）: 敌人护甲/魔抗 -X%
  Set 17: -14%

#### 减伤/护盾类
- **Bastion**（壁垒）: 团队护甲/魔抗
  Set 17: (2) 全队 12 / Bastions 16（首 10s 翻倍） · (4) 35 · (6) 55；非 Bastion +20
- **Vanguard**（先锋）: 开战时获得护盾
  Set 17: (2/4/6) 16/28/40% 最大生命护盾；护盾时获 5/-/10% 减伤
- **Brawler**（斗士）: 纯生命加成
  Set 17: (2/4/6) 全队 7% + Brawlers 25/45/65% max Health

#### 召唤类
- **Shepherd**（Set 17）: 召唤 Bia / Bayin，力量 = Shepherd 星级总和
  (3/5/7) 召唤 Bia / 召唤 Bayin / 联结加深
- **Primordian**（Set 17）: 受/造伤害时召唤 Swarmling
  (2/3) 按星级生成 Swarmling 数量递增 + 每回合随机送 1-2 费单位
- **Meeple**（Set 17）: 召唤 Meeps 伴随
  (3/5/7/10) 2%/3%/4%/6% 生命 + 125/250/400/1000 flat + 解锁克隆槽 + 10 件套召唤四大 MeepLord

#### 复活类
- **Yorick / Necromancer**（多 set）: 友军死亡后变召唤物
  经典设计：死亡 → Minion of Light 或灵体持续战斗

#### 回蓝/经济类
- **Conduit**（Set 17）: 团队法力回复
  (2/3/4/5) 全队 1/1/2/2% + Conduits 3/4/6/7% 额外
- **Channeler**（Set 17 另一版本）: +20% 额外法力获取

#### 特殊召唤/范围增益
- **Bulwark**（Set 17 — Shen 专属）: 放置一个 relic 给相邻友军 18% 护盾 + 20% 攻速
  **设计亮点**：单羁绊 1 件套，固定在 Shen 身上；与 relic 放置位置强耦合
- **Stargazer**（Set 17）: 每局随机绘制一个星座（7 种），在棋盘上形成特定 hex 图案并增益这些 hex 上的单位
  **设计亮点**：每把游戏不同的 hex buff 分布，迫使玩家动态调整摆位

#### 条件触发类
- **Anima**（Set 17 — 连败羁绊）: 每次战斗失败获得 Tech；Tech 累积 100 时解锁新 Anima 武器
  **设计亮点**：**损失 = 进度条**，把"连败"本身变成一个 scaling 机制
- **Timebreaker**（Set 17）: 输 → 获免费 reroll；赢 → 在 Temporal Core 存 XP
  **设计亮点**：胜败都有奖励但方向不同
- **Dark Star**（Set 17）: 黑洞吞噬生命 ≤10% 的敌人
  (2/4/6/9) 黑洞 · +30% AD/AP · 最强 Dark Star supermassive +2 小黑洞 · 10 件套 CONSUME EVERYONE
  **设计亮点**：低血处决阈值 + 等级解锁"巨大化"形态
- **Fateweaver**（Set 17）: 所有概率效果变 Lucky（检查两次取较好结果）
  (2/4) Lucky 概率 · 20% 暴击率 + 10% 暴击伤害（暴击也 Lucky）
  **设计亮点**：把"随机"本身作为羁绊主题
- **Replicator**（Set 17）: 技能以 X% 强度重复一次
  (2/4) 25% / 50%
  **设计亮点**：技能两连发（简单但对 AoE 技能极强）

#### 羁绊专属变身/变形
- **Mecha**（Set 17）: (3) 单位变身最终形态 +60% 生命；(6) +1 队伍位
  **设计亮点**：变身 = 可视化强度等级；多队伍位解锁
- **Party Animal**（Set 17 — Blitzcrank 专属）: 血 <45% 变无敌修复 15% HP/s
  **设计亮点**：硬编死战回复（血低才触发）
- **Space Groove**（Set 17）: 进入 Groove 状态获攻速+回血；(5) 每秒 Groove 叠 3% AD/AP；10 件套 "THE GROOVE"（未解锁满级效果）
  **设计亮点**：持续战斗越久伤害越高的 scaling 类型

#### 陷阱/风险类（历代）
- **Hellion**（Set 5）: 战斗开始时多召唤一个 Hellion
- **Underworld**（Set 2）: 持续 X 秒后全员获增益
- **Galaxy Hunter**（Set 17）: Zed 从 augment 获取，克隆 alive 时 +40% AD
- **Doomer**（Set 17）: 敌人第一次受伤 = 消耗其 Doom 偷 8% AD/AP 给 Vex
- **Divine Duelist**（Set 17 — Fiora 专属）: 治疗 15% 玩家造成的损伤；Fiora "永远" 一对一决胜
- **Oracle**（Set 17 — Tahm Kench 专属）: 每 3 回合给玩家奖励
- **Factory New**（Set 17 — Graves 专属）: 每战后可购买永久升级

### C3. Augments（海克斯增强 / 星界增益，Set 17 代表）

Augments 在 2-1 / 3-2 / 4-2 阶段提供选择。三档：Silver / Gold / Prismatic。

#### 战斗类（Combat）
- **Arcane Viktor-y**（Gold）: 战斗 8 秒后晕眩敌方全员 1.5s，之后每 20 秒重复
- **Ascension**（Gold）: 战斗 12 秒后全队 +35% Damage Amp
- **Glass Cannon II**（Gold）: 后排单位开战 80% HP 但 +30% Damage Amp
- **Charge Transfer II**（Gold）: 全队 +8% Damage Amp；80% 过量伤害作为魔法伤害传给最近敌人
- **Soul Awakening**（Prismatic）: 10 秒后进攻属性叠加并最终附加 true damage

#### 经济类（Economy）
- **Cognitive Tax+**（Silver）: 获得 12 金 + 3 经验
- **Money Monsoon**（Prismatic）: 立即 +7 金；之后每回合 +7 金
- **Hedge Fund+**（Prismatic）: +30 金；最大利息上限升至 10
- **Commerce Core**（Prismatic）: 立即获 16 次免费刷新；之后每回合 +2 次
- **Hustler**（Gold）: 不获利息，但每个玩家战斗回合开始 +2 金 + 2 XP

#### 成长类（Scaling）
- **Early Learnings**（Gold）: 全队 +8% AD/AP，每场玩家战斗后 +1%
- **Late Game Scaling**（Gold）: 每回合 +2 XP；5 费单位 +12% HP + 12% AS
- **Warlord's Honor**（Gold）: 每回合全队 +5% AD/AP（叠加 4 次）
- **Comeback Story**（Prismatic）: 按玩家缺失血量 全队 +4 HP + 0.4% AS

#### 羁绊/装备类（Trait/Item）
- **Hard Commit**（Prismatic）: 随机 Emblem；每阶段开始获一个该羁绊的 1 星英雄
- **Legion of Threes**（Gold）: 随机 Emblem；3 费单位和持有 Emblem 的友军 +200 HP + 12% AS
- **Portable Forge**（经典，Set 12+ 多次出现）: 选择 1 个 Artifact（强力独特装备）
- **Cybernetic Implants**（经典，Set 6.5+ 多次出现）: 持有装备的友军 +100 HP + 20% AD
- **Urf's Grab Bag**（经典）: 获得 4 件基础装备部件

#### 经典历代 Augments 设计范式总结
- **"条件增益"**: 满足某个布尔条件时永久 buff（如 Cybernetic 要求持装备）
- **"时间增益"**: 战斗持续时间触发效果（如 Ascension 12 秒）
- **"损失补偿"**: 奖励连败/奖励缺失血量（如 Comeback Story）
- **"开局独特选择"**: 一次性大 buff 或独家羁绊（如 Hard Commit）
- **"Artifact 池"**: 从 4 个独特物品中选一（如 Portable Forge）

### C4. 地图 / 阶段机制

Set 17 特殊：**Space Gods Marketplace**（取代传统 Carousel）

#### Set 17 Space Gods 机制
9 位神明取代传统转盘，在 stage 2-4 / 3-4 / 4-4 出现（每次两位供选）。选择同一位神 ≥2 次后，stage 4-7 解锁该神的祝福宝库（1 金即可刷新）。

9 位神的定位：
1. **Ahri（九尾狐）— God of Opulence**: 每回合 2 金 + 2 XP + 2 刷新（stage 4-7）
2. **Aurelion Sol**: 每阶段完成任务换奖励
3. **Ekko**: 延迟收益型奖励（几回合后生效）
4. **Evelynn**: 用血量换奖励（金币/升星/Emblem）
5. **Kayle**: 大量组件
6. **Soraka**: HP + 金币（利连败）
7. **Thresh**: 全随机（ultimate gamble）
8. **Varus**: 适合 reroll 阵容（单位/Duplicator/金）
9. **Yasuo**: 在棋盘上生成特殊 hex 增益

#### 通用阶段机制（历代）
- **Carousel（旋转木马）**: 玩家按血量反向顺序选装备英雄；Set 17 改为 Space Gods
- **PvE Rounds（PvE 回合）**:
  - **Minions（小兵）**: stage 1-1 开局
  - **Krugs（石甲虫）**: 1-4
  - **Wolves（狼）**: 2-5 或类似
  - **Raptors（迅猛龙）**: 中期
  - **Dragons（龙）**: 6-X（高阶 PvE boss）
  - Riot's Scuttle / 其他 set 专属 PvE
- **Hexes 增益**: 部分 set 在棋盘上生成特殊 hex（例：Set 17 Stargazer 星座 hex、Yasuo Space God hex）

#### 跨 set 持续出现的 meta 机制
- **Augments（增益）系统**: Set 6 开始引入，每场战斗三次选择
- **Hero Augments（英雄增益）**: 特定英雄专属增益（Set 8 引入，Set 17 仍有 8 个 Hero Augments，含 Zed 解锁）
- **Artifacts（神器）**: 跨 set 出现，通过 Portable Forge 或 Scuttle Puddle 获得的独特装备
- **Tomes of Traits（羁绊之书）**: 玩家可选择获得特定羁绊的 Emblem
- **Choncc's Treasure / 类似宝箱**: 在某些 set 的里程碑回合给玩家一次性奖励（类似 Set 17 的 god blessing treasury）
